Title: Campus Tour UI

Description: 
Frontend UI for campus tour web based application

Project Scope

This repository contains the frontend UI of the Campus Tour Assistance system.

Development is done incrementally, committing page by page and component by component.

Core UI Modules :Navigation bar, Search interface, Category pages, Filter components, Home and layout structure

Development Approach: Feature-specific branches, Small, focused commits, UI components built and committed independently, Main branch updated only after stable merges

Tech: React, Vite, Tailwind CSS
